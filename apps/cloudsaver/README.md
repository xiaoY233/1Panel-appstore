## 功能特性

### 🔍 多源资源搜索

+ 支持多个资源订阅源搜索
+ 支持关键词搜索与资源链接解析
+ 支持豆瓣热门榜单展示

### 💾 网盘资源转存

+ 支持115 网盘，夸克网盘，天翼网盘，123云盘一键转存
+ 支持转存文件夹展示与选择

### 👥 多用户系统

+ 支持用户注册登录
+ 支持管理员与普通用户权限区分

### 📱 响应式设计

+ 支持 PC 端与移动端自适应布局
+ 针对不同设备优化的交互体验
