## 产品介绍

API 设计、开发、测试一体化协作平台 Apifox 的自托管通用 Runner

## 安装说明

进入 [ApiFox Web 端](https://app.apifox.com/)，前往主窗口 `团队资源` >> `通用 Runner` >> `部署通用 Runner` 获取安装必需的参数
