# OpenLiteSpeed Web 服务器

## 描述

OpenLiteSpeed 是由 LiteSpeed Technologies 开发和版权持有的高性能、轻量级开源 HTTP 服务器。用户可以根据 GPLv3 许可证的条款自由下载、使用、分发和修改 OpenLiteSpeed 及其源代码。

这是 OpenLiteSpeed 源代码的官方仓库，由 LiteSpeed Technologies 维护。
