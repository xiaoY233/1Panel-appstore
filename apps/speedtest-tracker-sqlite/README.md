## 应用说明

Speedtest Tracker 是一个自托管的互联网性能跟踪应用程序，可针对 Ookla 的 Speedtest 服务运行速度测试检查。

由于 Speedtest 服务主要节点位于国外，所以在国内使用时可能会有一些问题，但是在国外使用时，Speedtest Tracker 是一个非常好的选择。
