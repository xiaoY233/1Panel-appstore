## 简介

轻量级网络 IP 扫描器，带有 Web GUI。特性： Translation:

+ 发现新主机时发送通知
+ 监控主机在线/离线历史
+ 保持网络中所有主机的列表
+ 将数据发送到 InfluxDB2 以创建 Grafana 仪表板
