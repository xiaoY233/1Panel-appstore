## 简介

### 项目

创建/编辑/更新/存档您的项目

### 版块

每个项目都有一个版块，您可以在其中自定义列以适应您的流程

### 问题

创建问题，编写 Markdown 描述并附加文件

### 工作流程与看板

在列之间移动问题

### 时间跟踪

跟踪在项目和特定问题上的时间

### 时间报告

按项目和时间段生成时间报告

### 主题

根据您的喜好自定义 UI
