# NZBGet

NZBGet 是一个高性能的Usenet下载工具，支持Web界面管理和自动化下载。

## 功能特性

- 高性能Usenet下载
- 支持Web界面管理（默认端口6789）
- 支持自动化脚本和RSS订阅
- 支持多平台和多架构
- 轻量级、资源占用低

## 使用方法

1. 部署后访问 `http://服务器IP:6789` 进入Web管理界面
2. 默认用户名/密码：`nzbget` / `tegbzn6789`
3. 配置Usenet服务器信息，开始下载
4. 下载目录：`./downloads`
