## 安装说明

> 首次安装请先安装环境，然后再安装爱影CMS
>
> 必须初始化完成后，才能使用爱影CMS
>
> 后台管理：默认账户
>
> 用户名： `admin`
> 密码： `123456`

初始化后，请先创建站点，否则前端页面无法正常显示，返回 `500` 错误
