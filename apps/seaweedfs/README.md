## 简介

SeaweedFS 是一个快速的分布式存储系统，适用于 Blob、对象、文件和数据湖，可存储数十亿个文件！Blob 存储具有 O(1)
磁盘寻道和云分层功能。Filer 支持 Cloud Drive、xDC 复制、Kubernetes、POSIX FUSE 挂载、S3 API、S3 网关、Hadoop、WebDAV、加密和纠删码。
