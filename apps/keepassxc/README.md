# KeePassXC

KeePassXC 是一款开源、跨平台的密码管理器，支持Web界面和多种平台。

## 功能特性

- 跨平台密码管理，支持多种操作系统
- 支持Web界面管理（默认端口3000）
- 数据库文件本地存储，安全可靠
- 支持浏览器集成和自动填充
- 支持多用户和多数据库
- 轻量级、资源占用低

## 使用方法

1. 部署后访问 `http://服务器IP:3000` 进入Web管理界面
2. 创建或导入密码数据库（.kdbx 文件）
3. 配置浏览器插件实现自动填充
4. 数据库目录：`./db`
