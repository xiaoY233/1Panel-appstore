- # 使用[PairDrop](https://pairdrop.net/)*发送*

  [**在网页浏览器中**](https://pairdrop.net/) 共享本地文件。
  灵感源自 Apple 的 AirDrop。Snapdrop
  的分支。

  ## 特征

  可在所有平台上运行的本地网络上的文件共享。

  - 一种可行、类似 AirDrop 的多平台解决方案。
    - 通过点对点连接将图像、文档或文本发送到同一本地网络上的设备。
  - 互联网传输
    - 加入临时公共房间，通过互联网轻松传输文件。
  - Web 应用程序
    - 适用于所有配备现代网络浏览器的设备。
