# 阅读3 Webview插件

阅读是一款提供网络文学搜索的工具，为广大网络文学爱好者提供一种方便、快捷舒适的试读体验。

## 应用说明

当前应用为 `阅读3` 的 Webview 插件，提供了一个简单的 Webview 容器，用于 `阅读3` 的订阅源书籍加载功能。
