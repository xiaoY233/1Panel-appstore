## 简介

Sonatype Nexus存储库是您所有内部和第三方二进制组，组件和包装的真理的唯一来源。将所有开发工具集成到集中式二进制存储库管理器中，以便您可以选择最佳的开源组件，优化构建性能和快速船舶代码，同时提高SDLC的可见度。
