## 功能

*   闪电般的生成速度
*   支持 GitHub Flavored Markdown 和大多数 Octopress 插件
*   一键部署到 GitHub Pages、Heroku 等
*   强大的 API，无限可扩展
*   数百个[主题](https://hexo.io/themes/) & [插件](https://hexo.io/plugins/)

## 快速开始

**安装 Hexo**

```bash
$ npm install hexo-cli -g
```

使用 [brew](https://brew.sh/) 在 macOS 和 Linux 上安装：

```bash
$ brew install hexo
```

**设置您的博客**

```bash
$ hexo init blog
$ cd blog
```

**启动服务器**

```bash
$ hexo server
```

**创建一篇新文章**

```bash
$ hexo new "Hello Hexo"
```

**生成静态文件**

```bash
$ hexo generate
```
