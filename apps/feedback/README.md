# 问题反馈平台 (替兔小巢)

由于兔小巢将于2024-11-30日之后，限制免费版的使用，所以开发了这个问题反馈平台。

## 环境支持

- MariaDB 10.2+

创建数据库后，初始化请导入 `init.sql`, 后续依次执行 `{version}.sql` 版本号从小到大执行。

数据库文件可以在安装目录下的 `sql` 目录下找到。

## 安装说明
