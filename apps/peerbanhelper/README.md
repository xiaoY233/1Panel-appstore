# PeerBanHelper

自动封禁不受欢迎、吸血和异常的 BT 客户端，并支持自定义规则。

PeerId黑名单/UserAgent黑名单/IP CIDR/假进度/超量下载/进度回退/多播追猎/连锁封禁/伪装检测。

支持 qBittorrent/Transmission/Deluge/BiglyBT/Vuze(Azureus)。

## Docker 镜像地址

可通过安装应用/修改应用参数时，在`高级设置`>`编辑 compose 文件`修改对应镜像。
