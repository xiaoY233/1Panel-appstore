## 简介

Shopware 提供由人工智能、无代码自动化、代理商务和以客户为中心的设计驱动的先进电子商务解决方案。每项功能都是为快速部署和长期影响而设计的，使您能够更快地适应并在每个接触点提供无缝互动。
